;-------------
; DECODING:
;
; Ratio:#% 		determines encounter frequency
; pid:<#> 		specifies the PID of the generated creature
; pid:<string> 		returns the PID of the creature by string, i.e., a code word
; Script:<#> 		specifies the script number attached to the PID object
; Dead - 		prefix indicates that the creature will be dead when the map loads
; Item:<#> 		gives an item PID to the creature's inventory
; Item:<string> 	gives an item PID to the creature's inventory by code word
; Item:(#)<string>	here # is the quantity of the item
; Item:(<string>) 	specifies a whole list of items
; If:<something> 	conditions that can be set as <,>,<=,>=,==,& as well as:
;    	on Player(<characteristic/property/skill/other>), Enctr(Num_Critters),
;    	Time, Global(<global variable number>), Rand(chance%)
; position=#		enemy placement relative to the player:
;	huddle		- huddled in a compact crowd
;	surrounding	- surrounding the player from all sides
;	straight_line	- in a straight line
;	double_line	- in a double line
;	wedge		- in a wedge formation
;	cone		- in a cone formation
; spacing:#		(?) likely the direction for NPC generation relative to the player (at #=5, full surrounding)
; Distance:# 		Forced distance for a specific creature (if possible)
; <type_#>(tile number) Forces placement on a specific tile
;    	the object listed at <type_#> (this means the last one if there were
;    	multiples of that type).
; TileNum:# 		Forces this object to appear at a given tile #
; Spatial:<string> 	specifies a spatial script (trigger)???
; 
;-------------
